# Installation

## System Requirements

- Node.js 23 or later
- npm or yarn

## Installation Steps

1. Clone the repository:

```bash
git clone https://github.com/muratsu/mcplens.git
cd mcplens
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run start
```

## Building for Production

To create a production build:

```bash
npm run compile
```
