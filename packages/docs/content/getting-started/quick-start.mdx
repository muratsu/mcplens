# Quick Start

## Overview

MCPLens provides a user interface for working with the Model Context Protocol (MCP). This guide will help you get started quickly.

## Basic Usage

1. **Launch the application**: After installation, run the app using `npm run start`

2. **Connect to MCP**: Configure your connection settings

3. **Explore Features**: Navigate through the interface to explore available features

## Next Steps

Once you're familiar with the basics, explore the [Features](/docs/features) section to learn about all available capabilities.
