# Features

## Core Features

### MCP Integration

MCPLens provides seamless integration with the Model Context Protocol, allowing you to:

- Connect to MCP-enabled models
- View and manipulate context elements
- Debug and analyze model interactions

### User Interface

- Intuitive Electron-based desktop application
- Cross-platform support (Windows, macOS, Linux)
- Modern, responsive design

## Advanced Features

### Debugging Tools

- Real-time monitoring of MCP connections
- Detailed logs and error reporting
- Performance metrics and analysis

### Customization

- Configuration options for advanced users
- Theming and appearance settings
- Keyboard shortcuts for efficiency
